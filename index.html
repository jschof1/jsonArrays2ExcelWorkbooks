JSON to XLSX Converter

How it works

1. Click "Upload JSON" to select your JSON file from your computer.

2. The content of your JSON file will be displayed in the "JSON Preview" section, allowing you to review it.

3. Once your JSON is loaded, click "Generate XLSX" to convert it to an Excel file. This will download the file automatically to your device.

Note: Each top-level array in your JSON will become a separate sheet in the XLSX file.
